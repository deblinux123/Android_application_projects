## 🛠️ Getting Started

### Prerequisites

- Android Studio (latest stable)  
- Kotlin 1.9+  
- Gradle 8+  
- Android SDK 33+  

### Clone the Repository

```bash
git clone https://github.com/linux/android-projects.git
cd android-projects
````

### Open a Specific App

1. Launch **Android Studio**
2. Open the desired app folder (e.g. `StoreDemo`)
3. Let Gradle sync and build
4. Run the app on an emulator or device

### 🧪 Running Tests

Each app includes **unit** and **UI tests**. Run them with:

```bash
./gradlew test
```
